HTML & CSS Tags Guide

Learn the most important HTML tags and CSS properties with examples.

Common HTML Tags

Below are some common HTML tags along with their descriptions and usage.

Tag Description
<h1> - <h6> Defines headings, from largest (h1) to smallest (h6).
<p> Defines a paragraph of text.
<a> Creates a hyperlink to another page.
<img> Embeds an image in the page.
<ul>, <ol>, <li> Defines lists (unordered, ordered, and list items).
<table>, <tr>, <td> Creates a table with rows and columns.
<form> Creates an interactive form for user input.
<input> Defines an input field inside a form.
<button> Creates a clickable button.
<div> Defines a division or section in an HTML document.

Common CSS Properties

  • color: Sets the text color.
  • background-color: Defines the background color.
  • font-size: Sets the size of the text.
  • margin: Controls space around an element.
  • padding: Controls space inside an element.
  • border: Adds a border around an element.
  • text-align: Aligns text inside an element.
  • display: Defines how elements are displayed (e.g., block, inline).

HTML & CSS Code Examples

Example of an HTML Form:


<form> 
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

Example of a CSS Button:

 
button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
}
